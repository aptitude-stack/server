# AGENTS.md

## Module README Maintenance

- Every module directory under `app/` must include a `README.md`.
- When code in a module changes, update that module's `README.md` in the same change.
- When a new module directory is added under `app/`, add a `README.md` for it immediately.
- When a module directory is removed or renamed, update or remove its `README.md` accordingly.
- Keep each module `README.md` concise and current with:
  - purpose/responsibility
  - key files and their roles
  - important contracts (ports, endpoints, models)
  - known placeholders or planned work
- Keep `app/README.md` as the index of all module READMEs.
